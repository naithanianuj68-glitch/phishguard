🛡️ PhishGuard: Mission Digital Safety
Tagline: Spot the scam. Save the heart. Secure the web.

🌟 The Mission
PhishGuard is a gamified simulator designed specifically for the Silver Generation (Seniors). In a world of AI voice clones and deepfakes, we empower elders to recognize digital threats through interactive, global scenarios.

🚀 Key Features
Global Awareness: 15+ scenarios covering India (SBI/KYC), USA (IRS), and Global (Netflix/Amazon) threats.

AI-Threat Protection: Dedicated training for AI voice cloning and Deepfake video scams.

Senior-First Design: High-contrast Dark Mode, large typography, and tactile visual feedback.

Spread the Shield: Integrated WhatsApp sharing to warn friends and family.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Variables & Keyframes), Vanilla JavaScript.

Persistence: Web Storage API (localStorage) for Leaderboards and Theme preferences.

Deployment: Vercel.

📈 Impact
PhishGuard turns fear into intuition, making the internet a safer place for those who built our past.
